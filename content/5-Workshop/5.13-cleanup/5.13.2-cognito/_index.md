#### Delete Amazon Cognito

1. Go to **Amazon Cognito Console** → **User pools**. Select the user pool `playwright-user-pool`, then click the **Delete** button. A **Delete user pool "playwright-user-pool"?** dialog appears. Check both prerequisite boxes — **Delete Cognito domain `ap-southeast-1dxmd3q6ee` that you assigned** and **Deactivate deletion protection** — then type `playwright-user-pool` in the confirmation field and click **Delete**.
![Cognito 1](/images/5-Workshop/5.13-cleanup/5.13.2-cognito/con1.png?featherlight=false&width=90pc)

2. Upon success, the system displays a green banner **"User pool "playwright-user-pool" has been deleted successfully."** and the **User pools (0)** list returns to **No resources**.
![Cognito 2](/images/5-Workshop/5.13-cleanup/5.13.2-cognito/con2.png?featherlight=false&width=90pc)
