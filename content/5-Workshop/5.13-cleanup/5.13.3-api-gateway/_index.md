#### Delete Amazon API Gateway

1. Go to **API Gateway Console** → **APIs**. Select the API `playwright-api`, then click the **Delete** button. A **Delete API** dialog appears, warning that this will permanently delete all resources and stages, and that API callers will be unable to invoke the API. Type `confirm` in the confirmation field and click **Delete**.
![API Gateway 1](/images/5-Workshop/5.13-cleanup/5.13.3-api-gateway/api1.png?featherlight=false&width=90pc)

2. Upon success, the system displays a green banner **"Successfully deleted API `playwright-api`."** and the **APIs (0/0)** list returns to **No API**.
![API Gateway 2](/images/5-Workshop/5.13-cleanup/5.13.3-api-gateway/api2.png?featherlight=false&width=90pc)
